| Name    | Description                         |
| ------- | ----------------------------------- |

## Getting Started

Install all dependencies from repository root,

```bash
yarn install
```

Start the application,

```bash
yarn dev
```

and visit http://localhost:4783

## Deploying the application

Build all dependencies from repository root,

```bash
yarn build
```
