### âœ¨ Apps

| Name     | Description                         |
| -------- | ----------------------------------- |
| `web`    | NextJs dragverse website            |
| `mobile` | React Native dragverse mobile app   |
| `embed`  | NextJs application for video embeds |
| `api`    | Cloudflare functions                |
| `og`     | Open graph meta tags generator      |

| Name             | Description                               |
